# 🧮  Calculator Project (AST-based)

## 📌  Описание
Это учебный проект калькулятора, реализованный с использованием:
- токенизатора (lexer)
- парсера (AST)
- вычислителя (evaluator)
- поддержки математических операций

---

## ⚙️  Возможности
- сложение, вычитание, умножение, деление
- степень (**)
- целочисленное деление (//)
- остаток (%)
- унарные операции (+, -)
- скобки
- поддержка приоритетов операций
